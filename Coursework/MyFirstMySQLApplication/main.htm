Introduction

MySQL is a relational database management system (RDBMS) which has more than 11 million installations.The program runs as a server providing multi-user access to a number of databases.

PHP and MySQL are essential components for running popular content management systems such as Expression Engine and some BitTorrent trackers.

Wikipedia runs on MediaWiki software, which is written in PHP and uses a MySQL database.

MySQL is the de-facto standard database for web sites that support huge volumes of both data and end users (like Friendster, Yahoo, Google).

Look at http://www.mysql.com/customers/ for an overview of companies using MySQL.
